#  Employee Management System (EMS)

A React-based Employee Management System that allows users to manage employee records with CRUD operations.

##  Live Demo

   https://ems-project-54o5.vercel.app

## GitHub Repository

  https://github.com/vjnithish17/EMS-project

##  Features

-  Add Employee
-  Edit Employee
-  Delete Employee
-  View Employee Details
-  Search Employees
-  Responsive UI

##  Tech Stack

- React.js
- React Router DOM
- Axios
- JSON Server / REST API
- CSS3

##  Installation

```bash
git clone https://github.com/vjnithish17/EMS-project.git
cd EMS-project
npm install
npm start
```
